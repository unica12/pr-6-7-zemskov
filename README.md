Отчет о проделанной работе
В рамках проекта была успешно выполнена верстка ключевых экранов мобильного приложения, в том числе:

Экран регистрации нового пользователя;
Экран авторизации с полями для ввода логина и пароля;
Главная страница интернет-магазина с каталогом товаров;
Раздел «Избранное» для сохранения понравившихся товаров;
Корзина с выбранными для покупки товарами

Использованные элементы интерфейса:
Для разработки применялись различные виджеты Flutter, включая текстовые элементы (Text),
карточки товаров (Card), поля ввода (TextField),кнопки (ElevatedButton, IconButton), 
элементы навигации (BottomNavigationBarltem), изображения (Image),списки товаров в виде сетки (GridView),
а также интерактивные элементы (GestureDetector).
Особенности реализации:
Все основные экраны были воспроизведены в соответствии с требованиями проекта.
Единственное отличие от исходного макета - отсутствие сплеш-экрана (заставки), вызванное техническими сложностями при импорте и отображении логотипа. 
Остальные элементы интерфейса полностью соответствуют задачам и обеспечивают удобство использования приложения.
